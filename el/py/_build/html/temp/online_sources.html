online_sources module¶

This module contains classes representing online sources for searches. These classes implement standard interfaces, so that the user can implement additional sources. To avoid repeating the documentation, the variables and the functions are documented once in a dummy class DummySource.

class online_sources.DummySource¶

The dummy source class used to document the variables and the methods that appear in the resources. Note that some of the methods are applicable only for sources containing BibTeX/PDF resources. Such cases are clearly indicated.

static afterLinkForBib(driver, page)¶

Compute the position in the given HTML page following the href element containing the direct link to the page with the sought BibTeX entry. Sources that do not provide BibTeX entries or in which the BibTeX entries appear on the HTML page containing the search results itself do not implement this function.

Parameters:
  • driver – The web driver for selenium. See get_resource.driver.
  • page – The HTML page returned by the search.
Returns:

A tuple of two elements. The first element is the possibly modified HTML page. The second element is the start position of the direct link to the page with the sought BibTeX entry. If such position is not found, then this element is -1.
static afterLinkForPDF(driver, page)¶

Compute the position in the given HTML page following the href element containing the direct link to the page with the sought PDF. Sources that do not provide PDFs do not implement this function.

Parameters:
  • driver – The web driver for selenium. See get_resource.driver.
  • page – The HTML page returned by the search.
Returns:

A tuple of two elements. The first element is the possibly modified HTML page. The second element is the start position of the direct link to the page with the sought PDF. If such position is not found, then this element is -1.
allBibLinks()¶

Compute the list of all links to BibTeX entries for the given HTML page. This method is used by handlers.handleNewHTML. :param page: The HTML page containing the BibTeX entries.

Returns:The list of links to BibTeX entries.
static bibEntry(driver)¶

Return the first BibTeX entry located on the current page of the given web driver. Sources that do not provide BibTeX entries do not implement this function.

Parameters:driver – The web driver for selenium. See get_resource.driver.
Returns:The first BibTeX entry located on the current page.
name = 'Dummy'¶

The name of the source that will appear in the dialogue windows.

queryAddress = 'https://dummy.com/search?q='¶

The prefix of the address for search requests in the source.

redirections = True¶

True if the source uses http redirections, which need to be followed when looking for the paper PDF, and False otherwise. This variable is only applicable to sources providing links to PDFs stored on third-party servers.

searchPageElementName = 'link-element'¶

The name of the HTML class of the element on the query page whose appearance signifies that the results of the search can be read. Sources in which no particular element needs to appear do not define this variable.
