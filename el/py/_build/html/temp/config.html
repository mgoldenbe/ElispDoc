config module¶

This module defines the following configuration variables, which are globally shared by all the modules:

config.BIB_AUTOMATED_SOURCES = ['DBLP']¶

The online sources used by the automated search for a paper’s BibTeX entry. The implemented sources are GoogleScholar, SemanticScholar and DBLP.

config.BIB_MANUAL_SOURCE = 'SemanticScholar'¶

The online sources used for manual download of a paper’s BibTeX entry in case the automated search does no produce the desired entry. The value None means that manual downloads should not be initiated automatically.

config.BROWSER = 'chrome'¶

The browser to be used for manual searches.

config.DROP_DIR = '<LiteRef root path>/drop/'¶

The folder for dropping the imported files and for the requests created by the command layer.

config.EVINCE_STARTUP_DELAY = 3¶

The delay in seconds to make sure that the PDF viewer opens before asking the user whether the found PDF is the desired one.

config.NOTES_DIR = '<LiteRef root path>/survey/'¶

The location for the surveys.

config.PAPERS_DIR = '<LiteRef root path>/papers/'¶

The location of the papers database.

config.PDF_AUTOMATED_SOURCES = ['GoogleScholar']¶

The online sources used by the automated search for a paper PDF. The implemented sources are GoogleScholar and SemanticScholar.

config.PDF_MANUAL_SOURCE = 'GoogleScholar'¶

The online sources used for manual download of a paper PDF in case the automated search does no produce the desired PDF. The value None means that manual downloads should not be initiated automatically.

config.ROOT_DIR = '<LiteRef root path>/'¶

The root folder of LiteRef. The value of this variable is read from the command line. Since the server is started automatically by the command layer, the user does not need to worry about supplying this value. He only needs to use the desired path in the Emacs configuration.
