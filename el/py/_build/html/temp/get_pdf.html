get_pdf module¶

This is the module documentation.

get_pdf.candidateBibFeedback(link, source)¶

Get the candidate BibTeX entry and ask the user whether it is the one. If LINK is empty string, the needed link is assumed to be already open.

get_pdf.candidatePDFFeedback(link, paperDir)¶

Download the candidate PDF and ask the user whether it is the one.

get_pdf.confirmPDF(fileName)¶

Open the given PDF and ask the user whether is the one he wanted.

get_pdf.entry2query(entry)¶

Compute a query based based on bibtex entry or search string.

get_pdf.firstLink(source, searchType, page)¶

Get the first link to the resource corresponding to searchType from the HTML source page. If the link is just the same page, return an empty string.

get_pdf.followRedirections(source, searchType, link)¶

Depending on source and searchType, follow redirections beginning from the given link until a direct link to the resrouce is obtained.

get_pdf.getResourceAutomated(entry, searchType, paperDir)¶

Obtain the resource for the entry in an automated manner.

get_pdf.getResourceManual(entry, searchType)¶

Obtain the resource for the entry in a manual manner.

get_pdf.pdfExists(entry, paperDir)¶

Check whether the needed pdf file is among the pdf files in the drop folder.

get_pdf.processCandidateBib(link, source)¶

Process the candidate PDF located at link.

get_pdf.processCandidatePDF(link, paperDir)¶

Process the candidate PDF located at link.

get_pdf.sourceQueryAddress(source, entry)¶

Get the address corresponding to the query from the given source.

get_pdf.sourceSearchResultsHTML(source, entry)¶

Return the HTML source of the page of search results.
