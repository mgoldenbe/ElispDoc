utils module¶

This module contains utility functions used in the other modules of the server.

class utils.ProgressBox(message)¶

The modal window showing the status of the current operation.

utils.center(toplevel)¶

Centers a window in TKinter.

The code is taken from https://stackoverflow.com/a/3353112/2725810. :param toplevel: The window to be centered.

utils.dirFiles(dirName, pattern)¶

Return the list of files in the given directory that satisfy the given pattern.

Parameters:
  • dirName – Name of the directory.
  • pattern – The pattern for the names of the files, such as ‘*.pdf’.
Returns:

The list of files.
utils.getClassElement(driver, className)¶

Wait for the first element of the given HTML class to appear and return it.

Parameters:
  • driver – The web driver for selenium. See get_resource.driver.
  • className – The name of the HTML class.
Returns:

The element of the requested HTML class.
utils.getIDElement(driver, id)¶

Wait for the first element with the given ID to appear and return it.

Parameters:
  • driver – The web driver for selenium. See get_resource.driver.
  • id – The requested element ID.
Returns:

The requested element.
utils.multisplit(s, delims)¶

Split the given string on the given delimeters.

The code is taken from https://stackoverflow.com/a/10655874/2725810.

Parameters:
  • s – The string to be split.
  • delims – The delimiters.
Returns:

The list representing the split string.
utils.readFile(fileName)¶

Return the contents of the file with the given name.

Parameters:fileName – The name of the file.
Returns:The contents of the file.
utils.wideYesNo(title, text)¶

Display a wide dialogue box for requesting a yes/no reply from the user and return the user’s reply.

Parameters:
  • title – The window’s caption.
  • text – The question.
Returns:

The user’s reply (yes or no).
